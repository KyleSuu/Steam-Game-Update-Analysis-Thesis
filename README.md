# Steam-Game-Update-Analysis-Thesis
# Thesis Code: Steam Game Update Features and Player review Sentiment Analysis

This project includes data acquisition, data cleaning, sentiment analysis (VADER) and visual code of various indicators for graduation thesis.

## Dataset Description
Due to GitHub's file size limit (the original data contains more than 1.6 million comments), the large dataset could not be uploaded directly to this repository.
Download the complete data for the project (including reviews_clean.csv, reviews_sentiment.csv, etc.) via the following Google Drive link:
https://drive.google.com/drive/folders/1U3XCqnXXNpUBYYsQ4d1p_zpd1UQYRNh5?usp=sharing

## How-to-run
After downloading the data, change the path of the 'OUT' variable in the Jupyter Notebook code to point to the folder where you store the data locally.
